# bad26-puzzle
---
## Requirements

- Installed `Node.js`

## Input

_`Please note, that to change input data you must edit 'data.txt' file.`_

**File should follow rules:**

- One puzzle per string
- Every string must contain only 6 symbols

## How to run

In project directory, where **`package.json`** located, run: 

```bash
npm run solve
```

## Output

All, results will be saved in files, which will be located located inside project in **`output`** sub directory.
